Frequently Asked Questions (FAQ)

What is Document Object Model (DOM)?

DOM or Document object model is a programming interface allows JavaScript to dynamically access and manipulate the structures and information of HTML and XML

How Do you select an element from DOM?

There is several ways to access elements Using DOM. Like, we can use "getElementById" , "getElementByClassName", "querySelector", "querySelectorAll", or by accessing children elements to parents and parent elements to children.

What is Event delegation in the context of DOM?, and why is it powerful?

Event delegation in context of DOM allows us to manage events at the higher level. Or parent level. This is powerful because it allows us to manage many events in higher level and and don't have to manage them individually. And that is efficient for memory.

How do you manipulate an element's attributes and styles using DOM?

We can Manipulate the elements attributes by using get and set attribute method. we also can remove attribute by Using "removeAttribute". We can manipulate the styles by using the "style" keyword.
